📦 Install FFmpeg
Step-by-step setup guide for every platform
🪟 Windows
🍎 macOS
🐧 Ubuntu / Debian
🎩 Fedora / RHEL
🔵 Arch / Manjaro
🤖 Android (Termux)
Method 1 — Winget Recommended · Windows 10/11

Built-in Windows package manager. No extra tools needed.

POWERSHELL / CMD (Admin)winget install Gyan.FFmpeg
✅ Run as Administrator. Winget automatically adds FFmpeg to your system PATH — no extra steps needed.
Method 2 — Chocolatey

First install Chocolatey from chocolatey.org, then run:

ADMIN CMDchoco install ffmpeg-full
💡 Use ffmpeg-full instead of ffmpeg to include all codecs (libx265, AV1, etc).
Method 3 — Scoop

Install Scoop from scoop.sh, then:

POWERSHELLscoop install ffmpeg
Method 4 — Manual Download
  1. Go to https://www.gyan.dev/ffmpeg/builds/
  2. Download ffmpeg-release-full.7z
  3. Extract to C:\ffmpeg\
  4. Open System Properties → Advanced → Environment Variables
  5. Under System Variables, find Path → Edit → New
  6. Add C:\ffmpeg\bin → OK → restart CMD
Add to PATH manually (if needed)
POWERSHELL (Admin)setx /M PATH "$env:PATH;C:\ffmpeg\bin"
Method 1 — Homebrew Recommended

Install Homebrew from brew.sh if you haven't already, then:

TERMINALbrew install ffmpeg
✅ Homebrew automatically adds FFmpeg to your PATH. Works on both Intel and Apple Silicon (M1/M2/M3) Macs.
Method 1b — Homebrew with all codecs

Includes non-free codecs like x265, AV1, FDK-AAC:

TERMINALbrew install ffmpeg --with-chromaprint --with-fdk-aac --with-libass --with-librsvg --with-libsoxr --with-libssh --with-tesseract --with-libvidstab --with-x265 --with-xvid --with-zeromq --with-zimg
Method 2 — MacPorts
TERMINALsudo port install ffmpeg +nonfree
Method 3 — Manual Download
  1. Go to https://evermeet.cx/ffmpeg/
  2. Download the latest static build for macOS
  3. Extract and move the binary: sudo mv ffmpeg /usr/local/bin/
  4. Allow it in System Settings → Privacy & Security if macOS blocks it
TERMINALsudo mv ~/Downloads/ffmpeg /usr/local/bin/ && sudo chmod +x /usr/local/bin/ffmpeg
Apple Silicon note
🍎 On M1/M2/M3 Macs, Homebrew installs to /opt/homebrew/bin/ instead of /usr/local/bin/. Make sure /opt/homebrew/bin is in your PATH by adding it to ~/.zshrc.
~/.zshrcecho 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
Method 1 — apt Recommended

Works on Ubuntu 20.04+, Debian 10+, Linux Mint, Pop!_OS, and all derivatives.

TERMINALsudo apt update && sudo apt install ffmpeg -y
⚠️ The default apt version may be outdated. Use the Snap or PPA method below for the latest build.
Method 2 — Snap (latest stable)
TERMINALsudo snap install ffmpeg
Method 3 — Static Build (latest, no install needed)

Downloads a self-contained binary — no package manager required.

TERMINALwget https://johnvansickle.com/ffmpeg/releases/ffmpeg-release-amd64-static.tar.xz && tar xf ffmpeg-release-amd64-static.tar.xz && sudo mv ffmpeg-*-amd64-static/ffmpeg /usr/local/bin/
Check installed codecs
TERMINALffmpeg -codecs 2>/dev/null | grep -E "DEV|DEA"
Method 1 — RPM Fusion Recommended

FFmpeg is not in the official Fedora repos due to codec licensing. Enable RPM Fusion first:

TERMINAL (Step 1 — Enable RPM Fusion)sudo dnf install https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm https://mirrors.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm -y
TERMINAL (Step 2 — Install FFmpeg)sudo dnf install ffmpeg ffmpeg-devel -y
Method 2 — RHEL / AlmaLinux / Rocky Linux
TERMINALsudo dnf install epel-release -y && sudo dnf install --enablerepo=epel ffmpeg -y
💡 On RHEL 9+, also enable CRB: sudo crb enable before installing EPEL packages.
Method 3 — Flatpak (app-level, not system-wide)
TERMINALflatpak install flathub org.freedesktop.Platform.ffmpeg-full
Method 1 — pacman Recommended

FFmpeg is in the official Arch repos — always up to date.

TERMINALsudo pacman -S ffmpeg
Method 2 — ffmpeg-full (AUR, all codecs)

Includes every optional codec: FDK-AAC, MPEG-1/2, libx265, etc. Install with your AUR helper:

TERMINAL (yay)yay -S ffmpeg-full
TERMINAL (paru)paru -S ffmpeg-full
Manjaro
TERMINALsudo pacman -Syu ffmpeg
💡 Manjaro packages may lag behind Arch by a few days. Use -Syu to update all packages at once.
Prerequisites — Install Termux
⚠️ Do not use the Play Store version of Termux — it is outdated. Install from F-Droid at f-droid.org or GitHub Releases.
Install FFmpeg in Termux Recommended
TERMUXpkg update && pkg install ffmpeg
✅ Termux's FFmpeg build includes most common codecs. It runs natively — no root required.
Grant Storage Access

To access your phone's videos and downloads folder:

TERMUXtermux-setup-storage
💡 After running, your storage is accessible at ~/storage/. For example, videos in ~/storage/dcim/ and downloads in ~/storage/downloads/.
Example — Compress a video on Android
TERMUXffmpeg -i ~/storage/dcim/Camera/video.mp4 -c:v libx264 -crf 28 -preset fast ~/storage/downloads/output.mp4
Useful Termux packages
TERMUXpkg install ffmpeg yt-dlp python
💡 yt-dlp is a popular download tool that uses FFmpeg for post-processing. Together they let you download and convert media directly on your phone.
✅ Verify Installation (all platforms)
TERMINAL / CMDffmpeg -version
✅ You should see the FFmpeg version, build date, and enabled libraries. If you get "command not found", FFmpeg is not in your PATH — double-check the installation steps for your OS.
